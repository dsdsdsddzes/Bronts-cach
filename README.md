# Bronts Cach - نظام إدارة المهام المصرفي

موقع متقدم لإدارة المهام بتصميم احترافي يشبه المصارف، مع نظام تسجيل دخول آمن وإدارة مهام شاملة.

## المميزات 🌟

✅ **نظام تسجيل دخول آمن** - إنشاء حساب جديد وتسجيل دخول آمن
✅ **إدارة المهام** - إضافة وتعديل وحذف المهام بسهولة
✅ **تصنيفات** - تنظيم المهام حسب الفئات (عمل، شخصي، مالي)
✅ **تاريخ الاستحقاق** - تحديد تواريخ محددة للمهام
✅ **إحصائيات** - عرض إجمالي وعدد المهام المكتملة والمتبقية
✅ **التصميم الاحترافي** - واجهة حديثة تشبه تصميم المصارف
✅ **متجاوب** - يعمل بسلاسة على جميع الأجهزة
✅ **تخزين محلي** - تخزين آمن للبيانات على جهازك

## البدء 🚀

### 1. فتح الموقع
- اضغط نقراً مزدوجاً على ملف `index.html`
- أو اسحب ملف `index.html` إلى المتصفح

### 2. إنشاء حساب جديد
1. اختر "إنشاء حساب جديد"
2. أدخل:
   - الاسم الكامل
   - البريد الإلكتروني
   - كلمة المرور
3. اضغط على "إنشاء حساب"

### 3. تسجيل الدخول
1. أدخل البريد الإلكتروني وكلمة المرور
2. اضغط "تسجيل الدخول"

### 4. إضافة المهام
1. في الصفحة الرئيسية، املأ نموذج "إضافة مهمة جديدة"
2. أدخل:
   - **عنوان المهمة** (مطلوب)
   - **الفئة** (عمل / شخصي / مالي)
   - **تاريخ الاستحقاق** (اختياري)
   - **الوصف** (اختياري)
3. اضغط "إضافة المهمة"

### 5. إدارة المهام
- **تحديد كمكتملة**: انقر على صندوق الاختيار على المهمة
- **عرض التفاصيل**: انقر على بطاقة المهمة
- **حذف المهمة**: افتح التفاصيل واضغط "حذف المهمة"
- **تصفية**: استخدم أزرار الفئات على الجانب الأيسر

## التقنيات المستخدمة 🛠️

- **HTML5** - البنية الأساسية
- **CSS3** - التصميم والواجهة الاحترافية
- **JavaScript** - المنطق والوظائف التفاعلية
- **LocalStorage** - تخزين البيانات محلياً

## كيفية عمل البيانات 🔒

- جميع البيانات تُحفظ في `LocalStorage` على جهازك
- لا يتم إرسال أي بيانات إلى خادم
- البيانات تظل آمنة على جهازك فقط
- حذف سجل المتصفح سيحذف البيانات

## نصائح مهمة ⚠️

1. **استخدم بريد إلكتروني قوي** - لتسهيل استعادة الحساب
2. **تذكر كلمة المرور** - لا يمكن استعادتها حالياً
3. **حفظ البيانات**: تأكد من عدم حذف سجل المتصفح
4. **استخدام متصفح واحد** - للحصول على أفضل تجربة
